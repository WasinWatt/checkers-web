# Checkers-web
A web ui for playing checkers against thai-checkers-zero. Using React as a static front-end and Flask as a server.

## Installation
For static front-end
```
cd static
yarn install
```
For server
```
pip install flask
pip install pymongo
pip install keras
pip install tensorflow
pip install h5py
pip install numpy
```
## Usage
Make sure you use Python3
```
cd server
python3 server.py
```
or
```
python server.py
```
